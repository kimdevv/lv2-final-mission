# 치과 예약 서비스

## 기능

### 진료
- [x] 진료를 예약할 수 있다.
  - 진료 종류를 선택할 수 있다.
  - 날짜를 선택할 수 있다.
  - 시간을 선택할 수 있다.
  - 이미 예약된 시간에는 추가로 예약할 수 없다.
  - 법정 공휴일에는 진료하지 않는다.

- [x] 특정 기간동안 예약된 모든 진료 내역을 확인할 수 있다.

- [x] 예약된 진료를 삭제할 수 있다.
  - 진료를 예약한 멤버만 그 예약을 삭제할 수 있다.

- [x] 예약된 진료를 수정할 수 있다.
  - 진료를 예약한 멤버만 그 예약을 수정할 수 있다.
  - 진료 타입을 수정할 수 있다.
  - 날짜 및 시간 수정은 불가능하며, 삭제 후 해당 시간대에 다시 예약해야 한다.

- [x] 예약의 상세 정보를 확인할 수 있다.
  - 진료를 예약한 멤버만 그 예약을 상세 조회할 수 있다.
  - 상세 조회 시 진료를 예약한 시간(createdAt)과 예약자가 함께 조회된다.

### 멤버
- [ ] 회원가입할 수 있다.
  - 회원가입 시 필요한 정보는 다음과 같다.
    - 아이디(username)
    - 비밀번호(password)
    - 이름(name)
- [ ] 로그인할 수 있다.
  - 아이디, 비밀번호를 통해 로그인한다
  - 로그인 성공 시 JWT를 발급한다.
